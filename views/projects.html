    Express Analytics

    Express analytics is an express middleware that lets you track visitors when they visit your site. The middleware does not add cookies to the client but uses their ip to ananymously track hits, unique hits, country, region, time zone etc. I'm still working on this and will add more features eventually. The npm package is available on npm directory as well.



    Vahem

    Vahem is a lodash like npm library that makes it easy for you to write code. The library lets you import functions that do some frequent and annoying tasks. I created this library for myself when I noticed myself being bored with tasks like validation. The library is still incomplete and has functions like validations, random generator, data formatter and array methods. Feel free to contribute and use this. Built with node js and typescript.
